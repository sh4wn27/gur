# Images for Field Notes

Drop image files into the matching subfolder. Filenames must match the slugs below.
Missing files just show the patterned placeholder — nothing breaks.

## programs/

- team-to-team.jpg      → Team-to-Team
- engineering-cs-courses.jpg → Engineering & CS Courses
- camps-workshops.jpg   → Camps & Workshops

## insights/
- One .jpg per article title, slugified
- e.g. "What we learned..." → what-we-learned-from-1-200-hours-of-mentor-pairings.jpg

## Format
Any web-safe format (.jpg, .png, .webp). Aspect ratios:
- programs: 3:4 (portrait)
- team: 4:5 (portrait)
- insights: 4:3 (landscape)
